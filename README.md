# Givana (Public Mirror)

This repository is a **read-only mirror** of the frontend portion of the private repo `juanwiley/givana-dev`.

- Backend code, API routes, database policies, secrets, and infra scripts are intentionally **not** included.
- For context and workflows, see **ASSISTANT_CONTEXT.md** in this repo.
- Issues and pull requests are disabled here; changes happen in the private repo and get mirrored.

If you’re collaborating: ask for access to the private repository or share patches against this mirror for review.
